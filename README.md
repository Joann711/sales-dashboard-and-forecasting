# Sales Dashboard and Revenue Forecasting

## Overview

This project combines Power BI and Python to analyze retail sales data and forecast future revenue trends.

## Dashboard Features

- Total Revenue KPI
- Total Units Sold KPI
- Total Transactions KPI
- Monthly Revenue Trend
- Revenue by Product Category and Gender
- Revenue Distribution by Gender
- Interactive Filters

## Tools Used

- Power BI
- Python
- Pandas
- Matplotlib
- Scikit-learn
